Step 1 — Goal
    The program should collect a user’s personal details and display a profile summary.

Step 2 — Inputs
The program will ask the user to enter:
    Name → text
    Age → number
    Favorite programming language → text

Step 3 — Output
    The program will display a formatted profile summary showing:
    Name
    Age
    Favorite language

Step 4 — Logic
    Ask the user to enter their name.
    Ask the user to enter their age.
    Convert the age from text to an integer.
    Ask for favorite programming language.
    Store all inputs in variables.
    Display the results using formatted print statements.

Step 5 — Flow
    Prompt user for inputs.
    Convert age to integer.
    Print the profile summary.
