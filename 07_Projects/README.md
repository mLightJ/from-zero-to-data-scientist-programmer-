# Student Performance Analysis

## Project Overview
Analysis of 1,000 student records to identify key factors 
influencing academic performance at a secondary school.

## Business Question
What factors most influence whether a student passes or fails 
— and what should the school do about it?

## Tools Used
- Python
- Pandas
- Matplotlib & Seaborn
- SciPy (Hypothesis Testing)

## Key Findings
- Students on standard lunch scored 7.66 points higher on average 
  than those on free/reduced lunch — confirmed statistically significant
- Test preparation improved average scores by approximately 8 points
- Parental education level showed a 10-point gap between highest 
  and lowest groups

## Recommendation
Offer free test preparation to all students, prioritizing those 
from lower income backgrounds. Introduce a parent engagement 
program to strengthen home learning support.
